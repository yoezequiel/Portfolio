## Overview
A modern, cloud-native Point of Sale system designed for fast-paced retail environments. It provides real-time sales tracking, inventory management, and detailed financial reporting across multiple locations, all through a responsive web interface.

## Technical Challenges
Real-time synchronization across multiple terminals was achieved using a combination of WebSockets and Turso's distributed database. Handling edge cases like concurrent transactions on the same SKU during peak hours required implementing optimistic concurrency control and a robust conflict resolution strategy.

## Results & Impact
The system currently handles hundreds of transactions daily with sub-second latency. The real-time reporting module has empowered managers to make data-driven inventory decisions, reducing waste by 20% and improving overall stock turnover rates.
