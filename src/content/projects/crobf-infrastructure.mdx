## Overview
The backbone of CROBF's digital presence, this project involved architecting a scalable cloud infrastructure to host institutional websites, private mail servers, and internal automation tools. It serves as the foundation for all organizational communications and digital assets.

## Technical Challenges
The main challenge was ensuring high availability for the mail server while maintaining strict security protocols (SPF, DKIM, DMARC) to prevent spoofing and ensure high deliverability. I also automated the deployment pipeline using GitHub Actions to ensure zero-downtime updates for the main website and associated microservices.

## Results & Impact
Achieved 99.9% uptime over the last year. The institutional communication has been streamlined, and the automated CI/CD pipelines have reduced deployment errors by 75%, allowing the team to focus on core development rather than infrastructure maintenance.
